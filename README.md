## 🌟 Giới thiệu
**KAIROS Quant System v3.0** là một hệ sinh thái giao dịch định lượng (Quantitative Trading) thế hệ mới, được thiết kế để giải quyết những thách thức phức tạp nhất của thị trường tài chính hiện đại. Với triết lý lấy dữ liệu làm trung tâm và AI làm động lực, hệ thống mang đến một quy trình đầu tư khép kín từ khâu nghiên cứu, thử nghiệm đến thực thi thực tế.

Hệ thống kết hợp sức mạnh của **Trí tuệ bầy đàn (Swarm Intelligence)** với các mô hình **Học máy (Machine Learning)** tiên tiến để nhận diện trạng thái thị trường và tối ưu hóa điểm vào lệnh, giúp nhà đầu tư đưa ra quyết định dựa trên dữ liệu thay vì cảm xúc.

## 🚀 Tính năng cốt lõi

### 1. Swarm Intelligence Engine (A03)
Hệ thống sử dụng mạng lưới các tác nhân AI (Multi-agents) có khả năng phối hợp nhịp nhàng:
- **Expert Agents**: Các chuyên gia AI phân tích đa chiều (Kỹ thuật, Chu kỳ, Vĩ mô).
- **Long-term Memory**: Khả năng ghi nhớ và học hỏi từ các kịch bản thị trường trong quá khứ.
- **Collaborative Decision**: Tối ưu hóa tín hiệu thông qua sự đồng thuận giữa các tác nhân.

### 2. Bộ công cụ Kiểm chứng (Backtesting)
- **High Performance**: Xử lý dữ liệu OHLCV lớn với tốc độ cao nhờ tối ưu hóa thuật toán.
- **Detailed Metrics**: Cung cấp đầy đủ chỉ số: Sharpe Ratio, Max Drawdown, Win Rate, Profit Factor...
- **Visual Reports**: Biểu đồ trực quan giúp theo dõi sự tăng trưởng của vốn (Equity Curve).

### 3. Phân tích trạng thái thị trường (ML Market Regime)
- Tự động phân loại thị trường: Trend (Tăng/Giảm) hoặc Sideway.
- Điều chỉnh tham số chiến lược theo thời gian thực để thích nghi với biến động.

### 4. Giao diện Dashboard hiện đại
- Dashboard Real-time sử dụng **Server-Sent Events (SSE)**.
- Theo dõi luồng tư duy của AI và các lệnh thực thi tức thời.
- UI/UX cao cấp, tối ưu cho việc theo dõi nhiều chiến lược cùng lúc.

## 📂 Cấu trúc thư mục

- `processing/`: Chứa lõi xử lý Backtest, ML và Nghiên cứu.
- `services/`: 
    - `agents/`: Hệ thống trí tuệ bầy đàn và tư duy AI.
    - `api/`: Server FastAPI cung cấp API cho Frontend.
    - `execution/`: Module thực thi lệnh và quản lý giao dịch.
    - `llm/`: Kết nối với các mô hình ngôn ngữ lớn.
- `frontend/`: Mã nguồn giao diện người dùng.
- `utils/`: Các công cụ tiện ích (Cấu hình, Ghi log, Đường dẫn).

## ⚙️ Cài đặt & Sử dụng

### 1. Yêu cầu hệ thống
- Python 3.12 trở lên.
- Node.js & npm (cho Frontend).

### 2. Cài đặt Backend
```bash
# Cài đặt thư viện
pip install -r requirements.txt

# Cấu hình môi trường (Tạo file .env từ template nếu có)
python khoi_dong_he_thong.py
```

### 3. Cài đặt Frontend
```bash
cd frontend
npm install
npm run dev
```
